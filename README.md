Установка зависимостей
bash
# Backend (Python)
pip install -r requirements.txt

# Frontend (Node.js)
cd frontend
npm install

Использование
Через Telegram бота:
Найдите бота в Telegram

Отправьте сообщение в формате:

text
Адрес места происшествия: [адрес]
Описание происшествия: [описание проблемы]
Через веб-интерфейс:
Откройте http://localhost:3000

Просматривайте список жалоб

Отмечайте жалобы как обработанные
